# 💊 MediCare Pharmacy Inventory System

A complete pharmacy inventory management system with real-time tracking, AI predictions, and dark mode.

## ✨ Features
- 🔐 Secure Login
- 📊 Real-time Dashboard
- 💊 Medicine Management
- ⚠️ Low Stock & Expiry Alerts
- 💰 Quick Sale
- 🤖 AI Predictions
- 🌙 Dark Mode

## 🛠️ Tech Stack
- React.js
- Spring Boot
- Supabase PostgreSQL

## 🚀 Quick Start
### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
./mvnw spring-boot:run

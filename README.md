# 💊 MediCare Pharmacy Inventory System

A complete pharmacy inventory management system with real-time tracking, AI predictions, and dark mode.

## ✨ Features
- 🔐 **Secure Login** - Pharmacist authentication
- 📊 **Real-time Dashboard** - Live stats & analytics
- 💊 **Medicine Management** - Add, edit, delete medicines
- ⚠️ **Smart Alerts** - Low stock & expiry notifications
- 💰 **Quick Sale** - 2-click dispensing
- 🤖 **AI Predictions** - Stock & expiry predictions
- 🌙 **Dark Mode** - Eye-friendly theme
- 📈 **Sales History** - Track all transactions

## 🛠️ Tech Stack
- **Frontend:** React.js, CSS3, React Icons
- **Backend:** Spring Boot, Java
- **Database:** Supabase PostgreSQL

## 🚀 Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
./mvnw spring-boot:run
```
